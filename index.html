JQuery Information

This is my website for the Summer Work at Blue Coat for Computer Science. I have used Javascript to:

  • Animate the title's colour using JQuery, by using the 
    $.animate
    function, and make it continuous by using a recursive function.
  • Animate the JQuery overview page with examples of the JQuery functions I listed.
  • Make a dinosaur run across the page when a viewer enters the konami code, which is:
    Up Up Down Down Left Right Left Right B A

with the following Javascript features:

  • Variables, such as 
    var x = 12; // global scope
    const x = 12; // global scope constant
    let x = 12 // block scope
    , these let you sustain values all through a function scope that you can change.
  • Conditionals, such as 
    if(x < 12) console.log(x)
    , these let you run code only if a value evaluates to a true value. I used this for the Konami code functionality to check that the array values are equal.
  • Functions, such as 
    function(x) return x + 2
    and (new arrow syntax) 
    (x) => x + 2
    , these let you run bits of code with values called arguments. These arguments can have default values in newer Javascript. I used these in the callbacks I passed to JQuery.
  • Objects, such as 
    { x: 20, y: 30 }
    , these let you associate multiple named values woth a single value. I used these to configure the JQuery animations.
  • Arrays, such as 
    [ 20, 12 ]
    , these let you associate multiple values with a single value.
  • TypedArrays, such as 
    new Int32Array([1, 3, 5, 7])
    , these let you store multiple values of a single type in a single array value. These are way faster than regular arrays because of their typed access but they are slightly more compilicated to work with. I used these to store the keyboard buttons pressed and the desired buttons for the Konami code functionality.

The rest of my website is about the JQuery I used.
